PRACTICA 1

Clonar el ejemplo

git clone https://gitlab.ci.gsnet.corp/asdigital/angular-ex2.git

Cambiar nombre

Cambiar nombre al directorio y en package.json

Instalar herramientas

npm install

Instalar dependencias js

bower install

Probar
grunt serve
grunt build
grunt serve:dist

Subir proyecto a Gitlab
Crear el repositorio en Gitlab dentro de un grupo para el proyecto
En local:
-Borrer el directorio .git
-Escribir los comandos que vienen en gitla al crear el repositorio. Ojo com git commit. Mejor git commit -m "Primer commit"
